# Tech Spec Examples

This directory contains real-world examples of technical specifications generated using this skill.

## Usage
- Examples serve as reference material for future specifications
- Claude can reference these examples to understand format and style preferences
- New examples should be added here as they are created and refined

## Organization
Examples will be organized by type:
- `features/` - Feature specifications
- `systems/` - System design documents
- `apis/` - API documentation
- `architecture/` - Architecture overviews

*Examples will be added as the skill is used to generate actual tech specs.*
